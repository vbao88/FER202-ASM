```typescript
interface StudentsState {

```

## 4. Đặc tả Redux

### File: `src/app/store.js`
- Tạo Redux store với `configureStore`
- Reducer: `students` từ studentsSlice

### File: `src/features/students/studentsSlice.js`

#### Async Thunks:
| Thunk | API call | Mô tả |
|-------|----------|-------|
| `fetchStudents` | GET /students | Lấy toàn bộ danh sách |
| `fetchStudentById` | GET /students/:id | Lấy 1 sinh viên |
| `addStudent` | POST /students | Thêm sinh viên |
| `updateStudent` | PUT /students/:id | Cập nhật sinh viên |
| `deleteStudent` | DELETE /students/:id | Xóa sinh viên |

#### Reducers (xử lý state):
- `pending`: `loading = true`, `error = null`
- `fulfilled`: cập nhật `items` / `selectedStudent`
- `rejected`: `loading = false`, `error = action.error.message`

#### Selectors:
```javascript
selectAllStudents(state)    // state.students.items
selectSelectedStudent(state)// state.students.selectedStudent
selectLoading(state)        // state.students.loading
selectError(state)          // state.students.error
```

## 5. Đặc tả Form Validation

### Form thêm / chỉnh sửa sinh viên:

| Field | Validation Rules |
|-------|-----------------|
| `name` | Bắt buộc, tối thiểu 2 ký tự, tối đa 100 ký tự |
| `studentId` | Bắt buộc, format: `SE\d{6}` (VD: SE180001) |
| `major` | Bắt buộc, tối thiểu 2 ký tự |
| `description` | Bắt buộc, tối thiểu 10 ký tự |
| `avatar` | Bắt buộc, phải là URL hợp lệ (bắt đầu bằng http/https) |
| `gpa` | Bắt buộc, số thực, 0.0 ≤ gpa ≤ 4.0 |
| `semesterScore` | Bắt buộc, số thực, 0 ≤ score ≤ 10 |

### Hiển thị lỗi:
- Lỗi hiển thị dưới mỗi field (inline error)
- Màu đỏ (#ef4444) với icon ⚠️
- Chỉ validate khi user đã touch field đó (onBlur) hoặc khi submit

### 7.1 Color Palette

```css
:root {
  --primary: #6366f1;       /* Indigo - màu chủ đạo */

  --secondary: #8b5cf6;     /* Violet */
  --danger: #ef4444;        /* Red - nút xóa */
  --success: #10b981;       /* Emerald - GPA cao */
  --warning: #f59e0b;       /* Amber - GPA trung bình */
  --bg-dark: #0f172a;       /* Slate 900 - nền tối */
  --bg-card: #1e293b;       /* Slate 800 - nền card */
  --bg-input: #334155;      /* Slate 700 - nền input */
  --text-primary: #f1f5f9;  /* Slate 100 */
  --text-secondary: #94a3b8;/* Slate 400 */
  --border: #475569;        /* Slate 600 */
}
```

### 7.2 Typography

```css
font-family: 'Inter', sans-serif; /* Google Fonts */
```

### 7.3 Design System
- **Dark mode** làm mặc định
- **Card**: glassmorphism effect (backdrop-filter blur)
- **Button primary**: gradient `--primary → --secondary`
- **Button danger**: solid `--danger`
- **Grid**: 3 cột (desktop) → 2 cột (tablet) → 1 cột (mobile)
- **Border radius**: 12px (card), 8px (button, input)
- **Transition**: 0.2s ease cho hover effects
- **Progress bar**: gradient màu theo mức GPA (đỏ/vàng/xanh)
